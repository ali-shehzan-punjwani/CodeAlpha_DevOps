# Task 1 – CI/CD Pipeline using Azure

## Objective

Build an automated CI/CD pipeline using Azure DevOps and Azure Container Registry.

## Project Description

This project demonstrates continuous integration using Azure Pipelines with Docker.

## Steps Completed

* Created Azure DevOps Organization
* Created Azure DevOps Project
* Connected GitHub Repository
* Configured Azure Pipeline
* Built Docker Image automatically
* Pushed Docker Image to Azure Container Registry

## Pipeline

GitHub Repository → Azure Pipeline → Docker Build → Azure Container Registry

## Challenge Faced

Deployment to Azure App Service could not be completed because the Azure subscription had a compute quota limit of **0 VMs**, preventing App Service Plan creation.

## Learning Outcomes

* Azure DevOps
* CI/CD Pipelines
* Docker Integration
* Azure Container Registry
* Pipeline Automation
